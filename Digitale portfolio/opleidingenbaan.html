Vorige school

Ik heb voor dat ik op deze opleiding kwam op de middelbare school de Lage Waard in Papendrecht gezeten op het niveau VMBO-TL.

Ik vond dit niet een hele leuke school want ik leerde niet echt veel wat ik nou kon toepassen in deze opleiding, maar ik heb er wel een gezellige tijd gehad. Ik vond het vak geschiedenis altijd heel interessant maar ik wilde hier eigenlijk niks in gaan doen.

De vakken Frans, aardrijkskunde en biologie heb ik als eerste laten vallen want dat vond ik de meest nutteloze vakken die er bestaan.

Baan

Als baan werk ik bij Landvast dat is een bioscoop/cultureel centrum in Alblasserdam. Ik vind het leuk om hier te werken. Ik leer hier niet heel veel wat ik kan toepassen bij de opleiding. Wel leer ik hier mijn sociale vaardigheden te verbeteren omdat je met veel mensen moet praten. Dit kan wel van pas komen op de stage.
